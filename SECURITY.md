#  보안정책

##  지원 버전

이 절을 사용하여 당신의 프로젝트의 어떤 버전이 있는지 사람들에게 말하십시오.
현재 보안 업데이트로 지원되고 있습니다.

|  버전  |  지원  |
| ------- | ------------------ |
|   5.1.x     | :white_check_mark: |
|   5.0.x     | :x:                |
|   4.0.x     | :white_check_mark: |
|   < 4.0     | :x:                |
